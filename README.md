# VuePagination

Вебсайт написанный на vue3, он берет данные с фейкового fast api и отображает его отсортированным с помощью пагинации.
Более того сайт включает в себя функционал для создания новых постов

## Стек технологий

- Html/css
- javaScript
- Vue 3
